## Spacing Scale

Declared values (must be multiples of 4):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, tab-pill internal gap (0.25rem) |
| sm | 8px | Compact element spacing, conversation list padding (0.5rem) |
| md | 16px | Conversation item padding, section padding, sidebar header gap (1rem) |
| lg | 24px | Sidebar header horizontal padding, messages area padding (1.5rem) |
| xl | 32px | Reserved for larger section gaps |
| 2xl | 48px | Reserved for major layout divisions |
| 3xl | 64px | Not used in this phase |

Exceptions: Tab pills container uses 4px (xs) internal padding with 8px (sm) pill padding. Sidebar width is fixed at 360px desktop, w-full on mobile. Detail panel placeholder is 320px (populated in Phase 4).

## Layout Specification

### 3-Column Shell (LAYOUT-01, LAYOUT-03)

```
+------------------------------------------------------------------+
| .chat-shell (flex, h-[calc(100vh-2rem)], m-4, rounded-2xl,      |
|              border border-border, bg-surface-container-low)      |
|                                                                    |
| +----------+-----------------------------+------------+           |
| | Sidebar  |       Chat Area             | Detail     |           |
| | 360px    |       flex-1                | 320px      |           |
| | min-w    |       min-w-0               | (Phase 4)  |           |
| |          |       bg-chat-thread-bg     | hidden     |           |
| +----------+-----------------------------+------------+           |
+------------------------------------------------------------------+
```

- Shell: `flex h-[calc(100vh-2rem)] m-4 rounded-2xl border border-border overflow-hidden bg-[--surface-container-low]`
- Sidebar: `w-[360px] min-w-[360px] flex flex-col border-r border-border-subtle bg-[--surface-container-low]`
- Chat area: `flex-1 min-w-0 flex flex-col relative bg-[--chat-thread-bg]`
- Detail placeholder: `hidden` (rendered in Phase 4; reserve the slot with `w-[320px]` behind a conditional)

### Mobile Responsiveness (LAYOUT-02)

- Below md (768px): sidebar takes full width when no chat selected; hides when chat selected
- Below xl (1280px): detail panel renders as Sheet overlay (Phase 4)
- Toggle: existing `selectedChat` state in useChatStore drives visibility
- Sidebar: `cn("h-full w-full md:w-[360px] shrink-0", selectedChat ? "hidden md:flex" : "flex")`
- Chat area: `cn("h-full flex-1 min-w-0", !selectedChat ? "hidden md:flex" : "flex")`

### Ambient Glow (LAYOUT-04)

Two pseudo-element radial gradients on the chat area:

1. Top-right: `radial-gradient(circle, rgba(139,92,246,0.04) 0%, transparent 70%)` -- 300x300px
2. Bottom-left: `radial-gradient(circle, rgba(139,92,246,0.02) 0%, transparent 70%)` -- 250x250px

Implementation: CSS `::before` and `::after` pseudo-elements with `pointer-events: none` and `z-index: 0`. Content sits at `z-index: 1`.

## Sidebar Specification

### Container (SIDE-01)

- Background: transparent (inherits from surface-container-low)
- Border-right: 1px solid rgba(255,255,255,0.06)
- No rounded corners (flush with shell edges)
- Internal padding: sidebar-header gets px-6 (1.5rem / lg) pt-6

### Header Section

Top-to-bottom vertical flow:

1. **Title row** (flex, items-center, justify-between):
   - Left: `<Heading level="page">Mensagens</Heading>` + subtitle "Comunicacao da equipe" in text-[0.6rem] uppercase tracking-[0.08em] text-muted-foreground/50 font-normal
   - Right: "Nova Conversa" button

2. **Search** (SIDE-03):
   - Use SearchInput component or equivalent: `bg-white/[0.04] border border-white/[0.08] rounded-lg px-3 py-2 pl-9 text-[0.7rem]`
   - Search icon: 14px, text-muted-foreground/55, absolute left
   - Placeholder: "Buscar conversas..."
   - Focus: `border-primary/30 ring-2 ring-primary/[0.08]`

3. **Tab pills** (SIDE-02):
   - Container: `flex gap-1 p-1 rounded-xl bg-white/[0.03] mx-6 mb-3`
   - Pills: Todas (default active), Privadas, Grupos, Processos
   - Each pill shows count in parentheses: e.g. "Todas (12)"
   - Active: `bg-primary/12 text-primary shadow-sm rounded-lg px-3 py-1.5 text-[0.7rem] font-normal`
   - Inactive: `text-muted-foreground/50 text-[0.7rem] font-normal`
   - Hover (inactive): `text-muted-foreground/70`
   - Count digit: `text-[0.6rem] tabular-nums opacity-50` (active: `opacity-60 text-primary`)

### "Nova Conversa" Button (SIDE-07)

- `bg-primary text-white rounded-xl px-3 py-1.5 text-[0.7rem] font-semibold`
- Icon: Plus (lucide), 14px, inline before text
- Box-shadow: `0 2px 8px rgba(139,92,246,0.25)`
- Hover: `bg-[#7c4ddb] translateY(-1px)`
- Triggers existing `novo-chat-dialog.tsx`

### Section Labels (SIDE-04)

- Text: "Fixadas", "Recentes"
- Style: `text-[0.6rem] font-semibold uppercase tracking-[0.1em] text-muted-foreground/35 px-3 pt-3 pb-1.5`

### Conversation List

- Container: `flex-1 overflow-y-auto px-2` with thin scrollbar (4px thumb, transparent track)
- Scrollbar thumb: `bg-white/[0.08] rounded-full`, hover `bg-white/15`

### Conversation Item (SIDE-05, SIDE-06)

```
+-----------------------------------------------+
| [Avatar 40px] [Name............] [Timestamp]   |
|              [Preview text.....] [Unread: 3]   |
+-----------------------------------------------+
```

- Container: `flex items-center gap-3 p-4 rounded-xl cursor-pointer transition-all duration-200`
- Default: `border border-transparent`
- Hover: `bg-foreground/[0.03]`
- Active: `bg-[--chat-sidebar-active] border border-primary/[0.08]`

**Avatar (40px):**
- Container: `relative w-10 h-10 rounded-xl overflow-hidden shrink-0`
- Image: `w-full h-full object-cover`
- Fallback: `w-full h-full flex items-center justify-center bg-primary/12 text-primary text-xs font-semibold` (initials)
- Online dot: `absolute -bottom-px -right-px w-2.5 h-2.5 rounded-full bg-success border-2 border-[--surface-container-low] z-10`
- Offline dot: `bg-muted-foreground/30` (same position)

**Info section (flex-1, min-w-0):**
- Top row (flex, items-center, justify-between, gap-2):
  - Name: `text-[0.8rem] font-semibold text-foreground truncate`
  - Timestamp: `text-[0.6rem] text-muted-foreground/40 tabular-nums shrink-0`
- Bottom row (flex, items-center, justify-between, gap-2, mt-0.5):
  - Preview: `text-[0.7rem] text-muted-foreground/50 truncate flex-1`
  - Unread badge: `min-w-[18px] h-[18px] rounded-full bg-primary text-white text-[0.6rem] font-semibold flex items-center justify-center px-[5px] shrink-0`

## Accessibility

| Concern | Implementation |
|---------|----------------|
| Keyboard navigation | Tab through search -> tab pills -> conversation list. Arrow keys navigate conversation items. Enter selects. |
| Focus visible | ring-2 ring-primary/20 on all interactive elements |
| Screen reader | Sidebar has role="navigation" aria-label="Conversas". Tab pills use role="tablist" with role="tab" and aria-selected. Conversation list uses role="listbox" with role="option". Unread badge uses aria-label="N mensagens nao lidas". |
| Reduced motion | Ambient glow and hover transitions respect prefers-reduced-motion: reduce |
| Color contrast | Conversation name (foreground on surface-container-low) meets WCAG AA. Unread badge (white on primary) meets WCAG AA. |
