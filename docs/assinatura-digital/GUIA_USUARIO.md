## 1. Introdução

Bem-vindo ao sistema de assinatura digital! Este guia vai te ajudar a preencher formulários de forma rápida e segura.

**O que você vai precisar:**
- CPF válido
- Documentos pessoais (RG, endereço, etc.)
- Câmera (se o formulário exigir foto)
- Permissão de localização (se o formulário exigir)
- Conexão com internet estável

**Tempo estimado:** 5-10 minutos

## 2. Acessando um Formulário

Você receberá um link como este:
```
https://seusite.com/formulario/trabalhista/procuracao-reclamante
```

**Dicas:**
- Use um navegador atualizado (Chrome, Firefox, Safari, Edge)
- Evite usar modo anônimo (pode bloquear câmera/localização)
- Prefira desktop/notebook para melhor experiência
- Mobile também funciona, mas pode ser menos confortável

## 3. Passo a Passo do Preenchimento

### Passo 1: Verificação de CPF

**O que fazer:**
1. Digite seu CPF (apenas números ou formatado)
2. Clique em "Continuar"

**O que acontece:**
- Sistema verifica se você já está cadastrado
- Se sim: seus dados serão carregados automaticamente
- Se não: você precisará preencher seus dados

**Dicas:**
- Digite apenas números (ex: 12345678900)
- Ou use formato com pontos e traço (ex: 123.456.789-00)
- Ambos funcionam!

**Erros comuns:**
- "CPF inválido": Verifique se digitou corretamente
- "Erro ao verificar CPF": Tente novamente em alguns segundos

### Passo 2: Dados Pessoais

**O que preencher:**
- Nome completo (como no RG)
- RG
- Data de nascimento
- Nacionalidade
- Estado civil
- Profissão
- Endereço completo
- Telefone
- E-mail

**Dicas:**
- **CEP:** Digite o CEP e o endereço será preenchido automaticamente!
- **Telefone:** Use formato (XX) XXXXX-XXXX
- **E-mail:** Confira se está correto (você pode receber confirmações)
- **Campos obrigatórios:** Marcados com asterisco (*)

**Atalho do CEP:**
1. Digite o CEP (apenas números)
2. Aguarde 1 segundo
3. Logradouro, bairro, cidade e UF serão preenchidos
4. Você só precisa adicionar número e complemento

**Erros comuns:**
- "CEP não encontrado": Verifique se digitou corretamente
- "Campo obrigatório": Preencha todos os campos com *
- "E-mail inválido": Use formato nome@dominio.com

### Passo 3: Formulário Dinâmico

Esta etapa varia conforme o formulário. Você pode ver campos como:

**Exemplos de campos:**
- Nome da empresa reclamada
- Modalidade de trabalho (CLT, Autônomo, etc.)
- Período de trabalho
- Valor da causa
- Descrição dos fatos

**Dicas:**
- Leia as descrições abaixo de cada campo
- Alguns campos aparecem/desaparecem conforme suas respostas
- Campos obrigatórios têm asterisco (*)
- Use placeholders como exemplo

**Campos especiais:**
- **Select (lista suspensa):** Clique e escolha uma opção
- **Radio (botões):** Clique em uma opção
- **Checkbox (caixas):** Marque uma ou mais opções
- **Textarea (texto longo):** Use para descrições detalhadas
- **Data:** Use o calendário ou digite DD/MM/AAAA

### Passo 4: Captura de Foto (se necessário)

**O que fazer:**
1. Clique em "Permitir" quando o navegador pedir acesso à câmera
2. Posicione-se na frente da câmera
3. Clique em "Capturar Foto"
4. Se não gostar, clique em "Tirar Novamente"
5. Quando estiver satisfeito, clique em "Continuar"

**Dicas:**
- Use boa iluminação
- Olhe para a câmera
- Evite fotos tremidas
- Fundo neutro é melhor
- Foto será redimensionada para 500x500px

**Erros comuns:**
- "Câmera não encontrada": Verifique se seu dispositivo tem câmera
- "Permissão negada": Clique em "Permitir" quando o navegador pedir
- "Foto muito grande": Sistema redimensiona automaticamente, não se preocupe

**Troubleshooting:**
- Se a câmera não funcionar, recarregue a página
- Verifique se outro aplicativo não está usando a câmera
- Tente outro navegador

### Passo 5: Captura de Geolocalização (se necessário)

**O que fazer:**
1. Clique em "Permitir" quando o navegador pedir acesso à localização
2. Aguarde alguns segundos
3. Sistema captura automaticamente suas coordenadas
4. Clique em "Continuar"

**O que é capturado:**
- Latitude e longitude
- Precisão (em metros)
- Data e hora

**Dicas:**
- Ative GPS/localização no dispositivo
- Aguarde até a precisão melhorar (< 50m é bom)
- Se a precisão estiver ruim, clique em "Tentar Novamente"

**Erros comuns:**
- "Permissão negada": Clique em "Permitir" quando o navegador pedir
- "Localização não disponível": Verifique se GPS está ativado
- "Timeout": Tente novamente, pode demorar em locais fechados

**Privacidade:**
- Sua localização é usada apenas para validação
- Não é compartilhada com terceiros
- Fica registrada no documento assinado

### Passo 6: Visualização de PDF

**O que fazer:**
1. Aguarde o PDF ser gerado (pode levar alguns segundos)
2. Revise todos os dados preenchidos
3. Se estiver tudo correto, clique em "Continuar"
4. Se encontrar erro, clique em "Voltar" para corrigir

**Dicas:**
- Leia com atenção!
- Verifique se todos os campos estão preenchidos
- Confira se não há erros de digitação
- Use zoom se necessário (Ctrl + Scroll)

**O que verificar:**
- Nome completo correto
- CPF correto
- Endereço completo
- Dados específicos do formulário
- Formatação do documento

### Passo 7: Assinatura Manuscrita

**O que fazer:**
1. Use o mouse (desktop) ou dedo (mobile) para assinar
2. Assine dentro da área branca
3. Se errar, clique em "Limpar" e assine novamente
4. Quando estiver satisfeito, clique em "Finalizar"

**Dicas:**
- Assine como você assina normalmente
- Não precisa ser perfeito
- Use traços firmes
- Evite assinaturas muito pequenas
- Evite assinaturas muito rápidas (pode ficar tremida)

**O que é capturado:**
- Imagem da assinatura
- Número de pontos
- Número de traços
- Tempo de assinatura
- Dimensões da assinatura

**Erros comuns:**
- "Assinatura vazia": Você precisa assinar antes de continuar
- "Assinatura muito pequena": Tente assinar maior

### Passo 8: Sucesso!

**O que você verá:**
- Mensagem de sucesso
- Lista de PDFs gerados
- Botões de download

**O que fazer:**
1. Clique em cada PDF para baixar individualmente
2. Ou clique em "Baixar Todos (ZIP)" para baixar tudo de uma vez
3. Salve os arquivos em local seguro
4. Se precisar, clique em "Preencher Novo Formulário"

**Dicas:**
- Baixe TODOS os PDFs
- Guarde em local seguro (Google Drive, Dropbox, etc.)
- Faça backup
- Não perca os arquivos!

**Nomes dos arquivos:**
- Formato: `[nome-template]_[seu-nome]_[data].pdf`
- Exemplo: `procuracao_joao-silva_2024-01-15.pdf`

## 4. Dicas e Boas Práticas

### Antes de Começar
- Separe todos os documentos necessários
- Use conexão estável (Wi-Fi é melhor que 4G)
- Reserve 10-15 minutos sem interrupções
- Use navegador atualizado

### Durante o Preenchimento
- Leia todas as instruções
- Não pule campos obrigatórios
- Revise antes de continuar
- Não feche a aba/navegador (pode perder dados)

### Após Finalizar
- Baixe TODOS os PDFs
- Verifique se os arquivos abrem corretamente
- Guarde em local seguro
- Faça backup

### Segurança
- Não compartilhe seus dados pessoais
- Não preencha formulários em computadores públicos
- Use conexão segura (HTTPS)
- Verifique se o link é legítimo

## 5. Problemas Comuns

### "Página não encontrada" (404)
**Causa:** Link incorreto ou formulário desativado
**Solução:** Verifique o link com quem te enviou

### "Erro ao carregar formulário"
**Causa:** Problema de conexão ou servidor
**Solução:** Recarregue a página ou tente mais tarde

### "Câmera não funciona"
**Causa:** Permissão negada ou câmera em uso
**Solução:**
1. Recarregue a página
2. Clique em "Permitir" quando pedir
3. Feche outros apps que usam câmera
4. Tente outro navegador

### "Localização não funciona"
**Causa:** GPS desativado ou permissão negada
**Solução:**
1. Ative GPS no dispositivo
2. Clique em "Permitir" quando pedir
3. Aguarde alguns segundos
4. Tente em local aberto (melhor sinal)

### "PDF não gera"
**Causa:** Dados incompletos ou erro no servidor
**Solução:**
1. Volte e verifique se todos os campos estão preenchidos
2. Recarregue a página
3. Tente novamente
4. Se persistir, contate o suporte

### "Não consigo baixar PDF"
**Causa:** Bloqueador de pop-ups ou problema de permissão
**Solução:**
1. Desative bloqueador de pop-ups
2. Permita downloads no navegador
3. Tente clicar com botão direito → "Salvar como"

### "Perdi meus dados no meio do preenchimento"
**Causa:** Fechou a aba ou navegador
**Solução:**
- Infelizmente, precisa preencher novamente
- Dica: Não feche a aba até finalizar!

## 6. Perguntas Frequentes

### Quanto tempo leva para preencher?
**Resposta:** Entre 5 e 15 minutos, dependendo do formulário.

### Posso preencher no celular?
**Resposta:** Sim! Mas desktop é mais confortável.

### Preciso criar conta?
**Resposta:** Não! Basta ter CPF.

### Meus dados ficam salvos?
**Resposta:** Sim, para facilitar preenchimentos futuros.

### Posso editar depois de finalizar?
**Resposta:** Não. Revise bem antes de assinar!

### Posso preencher para outra pessoa?
**Resposta:** Não. Cada pessoa deve preencher com seu próprio CPF.

### O que acontece se eu fechar a aba?
**Resposta:** Você perde os dados e precisa preencher novamente.

### Posso voltar para etapas anteriores?
**Resposta:** Sim! Use o botão "Voltar" em cada etapa.

### Preciso imprimir os PDFs?
**Resposta:** Depende. Consulte quem te enviou o formulário.

### Os PDFs têm validade jurídica?
**Resposta:** Sim! Incluem assinatura digital, IP, localização e timestamp.

### Posso usar o mesmo formulário várias vezes?
**Resposta:** Sim! Cada preenchimento gera novos PDFs.

### Como sei se o formulário é legítimo?
**Resposta:** Verifique o domínio (HTTPS) e confirme com quem te enviou.

### Posso preencher offline?
**Resposta:** Não. É necessário conexão com internet.

### Quanto tempo os dados ficam salvos?
**Resposta:** Indefinidamente, para facilitar preenchimentos futuros.

### Posso deletar meus dados?
**Resposta:** Sim. Entre em contato com o suporte.

### O que fazer se encontrar um bug?
**Resposta:** Anote o erro, tire print e contate o suporte.
